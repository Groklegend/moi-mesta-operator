Макет 1

Компактная таблица

Плотные строки, действия-иконки на наведении. Похоже на текущее, но аккуратнее и быстрее читается.

│ ⋮⋮ 💇 Салоны 30 ✎ 🗑 │

Список + редактор

Слева список, справа сразу форма редактирования. Без модалок. Самый быстрый для админа.

┌─ Список ─┬─ Редактор ────────┐
│ 🍽️ Рест. │ Название: Стом. │
│ 🦷 Стом.◆│ Иконка: [🦷] … │
│ 💇 Салон │ Порядок: [20] │
│ 🔧 Авто │ [Удал.] [Сохр.] │
└──────────┴───────────────────┘
